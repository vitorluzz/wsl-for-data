# wsl-for-data

Antes de tudo, atualize o seu sistema operacional
```bash
sudo apt update && sudo apt full-upgrade -y
```

Instale o git
```bash
sudo apt install -y git
```

```bash
git clone https://github.com/vitorluzz/wsl-for-data.git
cd wsl-for-data
```

Execute o script
```bash
chmod +x setup.sh
./setup.sh
```
